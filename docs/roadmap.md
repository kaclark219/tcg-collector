# Roadmap

## Phase 1: Manual Collection Tracker

- manual card search
- card detail screen
- add-to-collection flow
- basic collection list
- initial user-owned copy model
- first real persistence layer after schema decisions

## Phase 2: Binders And Moving Cards

- create/edit/delete binders
- assign entries to binders
- move entries between binders
- resolve split-versus-move behavior for duplicates and grouped quantities
- binder-focused browsing and summaries

## Phase 3: Card Catalog Import/Search

- choose upstream card catalog strategy
- sync or cache card metadata locally
- better search by name, set, collector number, and filters
- catalog refresh tooling

## Phase 4: Scan Upload And Scan Sessions

- camera capture or image upload
- scan session creation and status tracking
- image storage strategy
- raw and processed scan artifacts

## Phase 5: OCR-Assisted Candidate Matching

- pluggable OCR interface
- extract likely name, set, and collector number
- rank candidate cards from catalog matches
- user confirmation before collection save

## Phase 6: Visual Similarity / Better Recognition

- OpenCV-based crop improvements
- border/artwork comparison
- ensemble ranking with OCR plus visual features
- confidence tuning and false-positive reduction

## Phase 7: Polish Features

- auth and sync
- offline support
- wishlist and trade flows
- import/export
- better onboarding
- scan history review
- analytics and quality-of-life improvements
